# Node.js REST API – E-commerce Backend

Projeto desenvolvido durante o curso de **Criação de APIs RESTful com Node.js, Express, TypeScript e Firebase**.

Esta aplicação consiste em uma **API RESTful completa para gerenciamento de dados de um E-commerce**, podendo ser aplicada em cenários como:

- aplicativos de delivery
- sistemas de vendas online
- plataformas de marketplace
- sistemas de força de vendas

A API foi construída utilizando **boas práticas de arquitetura backend**, separando responsabilidades em **Controllers, Services e Repositories**, além de integrar serviços do **Firebase** para autenticação, persistência de dados e armazenamento de arquivos.

---

# Tecnologias utilizadas

- Node.js
- Express
- TypeScript
- Firebase
- Cloud Firestore
- Firebase Authentication
- Cloud Storage
- Google Cloud Functions
- Joi / Celebrate (validação de dados)
- Express Async Handler

---

# Principais conceitos aplicados

Durante o desenvolvimento deste projeto foram aplicados conceitos importantes de backend:

- Criação de **APIs REST / RESTful**
- Uso correto dos **métodos HTTP**
- Estruturação de projeto backend com **TypeScript**
- Organização de código com padrões:
  - Controller
  - Service
  - Repository
- Validação de dados enviados pelo cliente
- Tratamento centralizado de erros
- Autenticação de usuários
- Upload e gerenciamento de imagens
- Persistência de dados em banco NoSQL
- Deploy serverless

---

# Funcionalidades da API

- Cadastro de usuários
- Autenticação com Firebase Authentication
- Persistência de dados no Firestore
- Upload de imagens para o Cloud Storage
- Validação de dados da API
- Tratamento de erros
- Monitoramento via Google Cloud
- Deploy com Cloud Functions
